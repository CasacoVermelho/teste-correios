## PRÉ REQUISITOS
Para a execução desse código, é necessário ter Python e selenium instalados no computador, alem da versão atualizada do google chrome.
Para instalar o Python, favor seguir para [o site oficial](https://www.python.org/downloads/).

Para instalar a biblioteca do Selenium em Python, digite no terminal:
'pip install selenium'

## COMO EXECUTAR

-Entrar na pasta do projeto utilizando o terminal
-Utilizar o comando 'python main.py'

mensagens ao usuário serão exibidas no terminal durante a execução.
